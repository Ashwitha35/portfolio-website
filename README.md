# Personal Portfolio Website

This is my personal portfolio website developed using Flask, HTML, CSS, and JavaScript.

## Features
- About Me section
- Skills section
- Projects section
- Contact form
- Responsive design

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## Project Structure

portfolio-website/
│
├── app.py
├── templates/
├── static/

## How to Run

1. Install Flask

pip install flask

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
